🍔 Food Delivery Web App — FYP
A full-stack food delivery website built with MERN Stack (MongoDB, Express, React, Node.js) with online payment, admin panel and AI-powered food recommendations.

⚙️ Tech Stack

Frontend: React.js, CSS3
Backend: Node.js, Express.js
Database: MongoDB Atlas
Auth: JWT
Payment: Stripe
Image Upload: Multer
AI Recommendations: Claude API (Anthropic)


✨ Features

Browse & search food menu
Add to cart & place orders
User login / signup (JWT)
Online payment (Stripe)
Order history & status tracking
Admin panel — add/delete food, manage orders
🤖 AI Food Recommendations — personalized suggestions based on order history & preferences


🗄️ Database Collections
CollectionFieldsUsersname, email, password, cartDataFoodsname, price, image, categoryOrdersuserId, items, amount, status, payment

🤖 AI Recommendation System
Uses Claude API to analyze user's past orders and suggest relevant food items — making the experience personalized and smart.
